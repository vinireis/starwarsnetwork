## Getting started
### Requirements 
* Java 11+
* Maven 3+

### Invoking Spring Boot server
```shell
mvn spring-boot:run
```

### Link to access Swagger-UI local
- [Swagger Local - http://localhost:8080/starwarsnetwork/app/swagger-ui/index.html](http://localhost:8080/starwarsnetwork/app/swagger-ui/index.html)
